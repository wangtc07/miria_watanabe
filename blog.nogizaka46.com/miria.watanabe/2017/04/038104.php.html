2017/0415Sat渡辺みり愛調和の加減

 
 
 
 
 
 
 
 
 
こんばんは、ナカダカナシカです。
 
 
 
 
 
 
 
 
 
 
 
 
 
嘘です、ワタナベミリアです。
( 花奈さんごめんなさい )
 
ネタバレしちゃうの早すぎかって
今自分で思っちゃいました
 
 
 
改めまして、渡辺です。
 
 
 
 
先日は幕張にて握手会がありました！
来て下さった皆様、ありがとうございます
 
 
 
1.2部
 
 
 
 
1部だけ限定で普段から使っています
ガチ眼鏡をかけてやっちゃいました〜
コンタクトは疲れちゃいますものね
 
 
 
3.4部
 
 
 
 
ポーズが何とも言えず気持ち悪いですね。
 
 
 
というか久しぶりのツインテール
やってしまいました、、
しかも短いから尚幼く見える(._.)
 
 
 
 
 
お花も！ありがとうございました！！
私の大好きなマイラーバルーンがたくさん
嬉しき。
 
 
とりあえず隠れてみました
 
 
 
 
 
初めましての皆様
毎回来て頂いている皆様
お久しぶりの皆様
 
色々なお話しが出来て笑ったり、感動したり
また頑張ろう！と力を頂きました
来て下さった皆様ありがとうございました
 
 
 
 
 
 
 
最近、生駒さんと見間違えると
いろいろなメンバーから言われます
生駒さんも言われているみたいで。
 
 
髪型や髪色、身長がだいたい同じだから
双子や双子やと2人で言い合ってます(^-^)v
生駒さん、ラブ(^-^)v
 
 
生駒さん、ラブ(^-^)v
 
 
 
 
( なんで2回言ったかは突っ込まないでね )
 
 
 
 
 
 
 
 
 
 
 
ライブまであと5日と迫ってきました
広い会場を12人だけで
皆様と一体感をつくりだせるのか
 
時間がない
 
皆きてくれるかな
 
 
 
 
不安をあげたら切りがない
　
 
 
研究生時代に鍛えてきた精神力が
あったはずなに情けないです
 
 
 
ライブまでの残された時間を
皆様に楽しんでいただけるよう
日々12人で頑張りますので宜しくお願いします
 
 
 
 
 
 
 
風船は生きている
 
 
 
 
 
 
 
 
またみてね
 
 
 
 
みり愛
