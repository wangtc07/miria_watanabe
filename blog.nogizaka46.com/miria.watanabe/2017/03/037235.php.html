2017/0307Tue渡辺みり愛涙ぐむ

 
 
 
 
 
 
こんばんは、渡辺です
 
 
 
 
 
先日、幕張メッセにて握手会が
ありました。
来て下さった皆様、ありがとうございました
 
 
 
1部
 
 
 
 
2.3部
 
 
 
 
 
 
 
 
どちらも好評で嬉しかったな〜
2部終わりには絢音の生誕祭にて
お手紙を読ませてもらいました
 
 
お手紙書くことは前日に分かっていたのに
そのままぴーすかぱーすか寝てしまいまして
朝起きた時、『あ、やってもうた』と
思いましたね。
 
急いで車の中で携帯のメモ書きにメモして
携帯を読みながら挑みました
グダグダになってしまい、ごめんなさい...
 
 
 
 
 
 
ちなみに楽屋もお隣さん、
気付けば自然と隣にいます
いつもありがとう、そしておめでとう。
 
 
 
 
また今週にも握手会あるなんて
テンションフィーバーって感じですね
( いやどんな感じだよ )
 
ウェルカムホームな感じで
お待ちしています
 
 
 
 
 
 
 
定期的に整体に行っているのですが
混んでて予約のとれないところ
に行きたくて、やっと取れたんですよ
なので行ってみたところ
 
「背中触らせていただいたんですけど触った感じ心臓の後ろらへんのここの筋肉がかったいですねー。心臓の後ろなのでかなり大事ですが、固いのでストレスを感じていて疲れてるんじゃないかなと思います」
 
 
ここまではいいんですよ
 
 
 
「ズズズーと下に行くとここの筋肉も固いですね、ここは肝臓の後ろなのでここが固いのは、人の目を気にし過ぎていたり周りのことに気を配り過ぎているんじゃないかなあ？それの神経の疲れというか、ストレスがここにくるのね。」
 
 らしいです。
 
 
 
 
 
 
 
 
 
 
 
 
 
占い師かよ、、
 
 
って思った1日でした
 
 
 
 
 
 
 
ここで問題です。
 
今回のポーズを相談したこの方は誰でしょうか？
 
 
( 難しすぎるとか言わないでね )
 
 
 
LOVEberry 発売中です。
飛鳥さん、絢音、蓮加ちゃん、みり愛で
出ています。ぜひ。
 
 
また見てね
 
 
 
 
 
みり愛
