2015/0928Mon渡辺みり愛三つ編みブーム

 
 
 
 
 
 
 
 
 
肩こりがとてもひどいです( ¨̮ )
かぼちゃスープ美味しいです( ¨̮ )
毛布が大好きです( ¨̮ )
髪の毛切ったの分かりますか( ¨̮ )
 
もうすぐ冬が近付いてくるから
すごく楽しみです。
 
 
 
 
 
先日は横浜にて握手会がありました
来てくれた皆さんありがとうございました！
 
12.13日は体調不良でお休み
してしまい振替のためにわざわざ
起こし下さった皆さんすみません。
 
 
 
皆さんが笑わせてくれたり
心配してくれたり本当に本当に
嬉しかったです(o^^o)
 
 
 
 
 
2.3部はこの服を着ました〜
1部の服は1枚目の写真だよ〜
 
 
 
10月18日はあれを着るよ♪
何だと思いますか...
似合うか不安だけど...
 
 
 
 
 
そしてアンダーメンバーは
TOUHOKU DREAM COLLECTION
にゲストとして出させて頂きました！  
 
 
モデルとしては飛鳥さん、
川後さん、伊織が出ました♪
 
かっこよかったな〜
 
 
 
 
 
髪型は最近好評のハーフツイン
させていただきました。
 
 
 
未央奈とあるお揃いの何かを買ったよ♡
みんな楽しみにしててね。
私が楽しみだな〜（笑）
 
 
 
 
 
 
こないだ仲のいい友達と久しぶりに
会うことが出来て嬉しかったんだけど
その時にテンションが上がって
早く歩いて
 
私「きょほうきょほう！」
 
友達1「きょほう？」
 
私「うんきょほう！」
 
友達1「何きょほうって？」
 
私「え？早く歩く競技だよ？」
 
友達1「それほこうだよ。きょほうってぶどうじゃん」
 
私「うわ！笑そうだった！笑」
 
 
 
もう1人の友達と待ち合わせて
３人でかいわしてるとき
 
 
 
私「またきょほうしよ（笑）」
 
友達1「ほこうねそれ（笑）」
 
私「あ、そうだほこうだわ（笑）」
 
友達2「何の話？」
 
私「早く走る競技の名前をさー、きょほうって言ってたよ〜（笑）」
 
友達 1「だからほこうだよっていう話をさっきしてたの（笑）」
 
友達2「それ......」
 
私・友達1「？？」
 
 
 
 
 
友達2「競歩(きょうほ)ね。」
 
 
 
 
 
私・友達1「...............( ¨̮ )」
 
 
 
結局どっちも間違えてて友達2の子が
冷静に直してくれました...(>_<)
 
 
 
 
 
 
 
新制服✨
 
今度は全身とかも撮ってくるね！
 
 
 
別れ際、もっと好きになるの衣装は
柄がとても好きです。
こういう洋服があれば即買いです。
 
 
 
 
 
 
9月誕生日の方
 
字が小さくて見にくかったら
ごめんなさい(>_<)
 
 
 
イジリーさん、蘭世。
お誕生日おめでとうございます！
 
 
 
 
 
 
また見てね
