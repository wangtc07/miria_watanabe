2015/0906Sun渡辺みり愛はちみつレモン

 
 
 
 
 
 
 
 
 
全国ツアーが無事終わりました(o^^o)
 
今回の全国ツアーで初めてみんなで
同じ衣装を着て踊った事が些細なこと
だけど自分的には凄く嬉しかったんです。
 
 
前回までは研究生という枠だけ
いつも違う衣装で登場してて、
複雑に思ってる自分がいました。
 
アンダーに昇格して太陽ノックの
衣装をみんなで着て踊れて
心も楽しくなりました〜
 
 
 
 
ライブ前の前日は毎回恒例の
皆さんからのコメントを
読んで助けられてました！
 
 
神宮に来てくれた方雨大丈夫でしたか〜？
風邪ひいてないですか（ *_* ）
その次の日学校やお仕事の方も
ライブに来てくださり
ありがとうございました♪
 
 
 
今回のツアーで無事みんな怪我せずに
突っ走れたことが本当に嬉しいです
それはスタッフの皆様、ファンの皆様
のおかげだと思っています(>_<)
 
本当にありがとうございました！
 
 
 
 
 
深川さんと抱きつき合うのが
最近の日課になってます( ¨̮ )
 
 
 
 
 
 
 
お花ありがとうございました〜！
 
 
 
 
 
 
 
 
 
 
NOGIBINGO!5見て頂けましたか？
浴衣でカラオケ大会〜
秋元シスターズで夏祭りを歌わせて
頂きました＼(^^)／
 
 
 
あの時声がガラガラでハスキーな
声だったのですが...
あの歌パートはなんとか頑張りました...（笑）
 
どうでしたかね...？
 
 
 
 
 
そしてアンダーライブ10月と12月
決まりました✨
 
また歌ったり踊ったり出来る...！
嬉しい！
 
 
 
 
そして12月はなんと武道館。
嬉しい気持ちもありますが不安も
あります。
 
人が埋まるかなあ？と心配ですが
まずは来てくれた方一人一人を
楽しいと思わせられるようなライブに
したいなあと思っております！
 
 
 
10月15〜25日まで、AiiAシアター
(18.19除く)
12月17.18日は武道館
 
お待ちしています( ¨̮ )( ¨̮ )
 
 
 
 
来てくれないと
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
拗ねます( ¨̮ )
 
 
 
 
 
 
 
◎告知◎
 
・ソニレコ!暇つぶしTV
 
今月のお当番は渡辺です！
この時も治りかけでハスキーな声に
なってます(>_<)
 
お聞き苦しいかもしれませんが
見てください！
 
 
 
 
 
 
また見てね
 
 
 
 
 
 
 
みり愛
